# Cinemania - Онлайн-каталог фильмов

![Cinemania](img/cinema-icon.png)

## Описание проекта

Cinemania — это современное веб-приложение для поиска, просмотра и каталогизации информации о фильмах. Приложение использует The Movie Database (TMDB) API для получения актуальных данных о тысячах фильмов, а также позволяет пользователям добавлять собственные фильмы в каталог.

Проект разработан в рамках курсовой работы по дисциплине «Проектирование веб-систем».

## Функциональные возможности

- 🔍 **Поиск фильмов** по названию и описанию с оптимизацией запросов (дебаунсинг)
- 📊 **Сортировка** по популярности, рейтингу, дате выхода и названию
- 📄 **Пагинация** с настраиваемым количеством элементов на странице (5, 10, 15, 20)
- 🎬 **Просмотр детальной информации** о фильме:
  - Полное описание сюжета
  - Информация о съёмочной группе (режиссёр, актёры)
  - Рейтинг, бюджет, сборы
  - Ссылки на трейлер, IMDb и официальный сайт
- ✏️ **Добавление пользовательских фильмов** с сохранением в локальном хранилище
- 🗑️ **Удаление добавленных фильмов**
- 📱 **Адаптивный дизайн** для мобильных, планшетных и десктопных устройств
- 🌓 **Поддержка тёмной темы** (на основе системных настроек)
- ⚡ **Индикация загрузки** и обработка ошибок

## Технологический стек

- **Язык программирования:** TypeScript
- **Сборщик:** Vite
- **HTTP-клиент:** Axios
- **Внешний API:** The Movie Database (TMDB)
- **Стилизация:** CSS3 с CSS-модулями и медиа-запросами
- **Архитектура:** ООП, модульный подход
- **Система контроля версий:** Git

## Требования для запуска

- Node.js версии 16 или выше
- npm версии 8 или выше
- Современный веб-браузер (Chrome, Firefox, Safari, Edge)
