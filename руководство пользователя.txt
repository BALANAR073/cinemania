## Введение

**Cinemania** — это веб-приложение, предназначенное для поиска, просмотра и каталогизации информации о фильмах. Приложение получает актуальные данные из базы The Movie Database (TMDB), а также позволяет пользователям добавлять собственные фильмы в локальный каталог.

Приложение разработано с учётом современных стандартов веб-разработки и обеспечивает комфортную работу как на настольных компьютерах, так и на мобильных устройствах.

## Системные требования

Для работы с приложением необходим современный веб-браузер с поддержкой JavaScript. Рекомендуемые браузеры:

| Браузер | Минимальная версия |
|---------|-------------------|
| Google Chrome | 90+ |
| Mozilla Firefox | 88+ |
| Microsoft Edge | 90+ |
| Apple Safari | 14+ |

Для полноценной работы требуется активное подключение к интернету, так как данные загружаются с внешнего API.

## Начало работы

При первом открытии приложения вы попадаете на главную страницу, где отображается каталог популярных фильмов. По умолчанию показывается 10 фильмов на странице.

![Главная страница](screenshots/main-page.png)

Основные элементы интерфейса:

1. **Шапка сайта** — содержит логотип и навигационное меню
2. **Строка поиска** — для поиска фильмов по названию или описанию
3. **Панель управления** — элементы сортировки и настройки отображения
4. **Сетка фильмов** — карточки с краткой информацией о фильмах
5. **Пагинация** — навигация по страницам каталога
6. **Подвал сайта** — контактная информация и ссылки

## Главная страница

Главная страница является центральной точкой входа в приложение. Здесь отображается каталог фильмов с возможностью поиска, сортировки и навигации.

### Карточка фильма

Каждая карточка содержит:

- **Постер фильма** — изображение, при отсутствии показывается заглушка
- **Название фильма** — на русском языке (если доступно)
- **Год выхода** — год премьеры фильма
- **Жанры** — до двух основных жанров
- **Краткое описание** — первые 120 символов сюжета
- **Рейтинг** — оценка по 10-балльной шкале (отображается в правом верхнем углу)
- **Количество голосов** — число пользователей, оценивших фильм

При наведении на карточку она слегка приподнимается, а для пользовательских фильмов появляется кнопка удаления (иконка корзины).

## Поиск фильмов

### Выполнение поиска

1. Введите поисковый запрос в строку поиска, расположенную под заголовком страницы.
2. Поиск выполняется автоматически после небольшой паузы (500 мс) после прекращения ввода.
3. Также можно нажать клавишу **Enter** или кнопку поиска (иконка лупы) для немедленного выполнения запроса.

### Что ищется

Поиск выполняется по следующим полям:
- Название фильма (включая оригинальное название)
- Описание (сюжет) фильма

Поиск нечувствителен к регистру букв.

### Результаты поиска

- Найденные фильмы отображаются в сетке
- Сортировка и пагинация применяются к результатам поиска
- Если фильмы не найдены, отображается соответствующее сообщение

### Сброс поиска

Чтобы вернуться к полному каталогу, очистите строку поиска. Каталог обновится автоматически.

## Сортировка и пагинация

### Сортировка

Для изменения порядка отображения фильмов используйте выпадающий список **«Сортировка»** на панели управления.

Доступные варианты сортировки:

| Вариант | Описание |
|---------|----------|
| По популярности ↓ | От наиболее к наименее популярным |
| По популярности ↑ | От наименее к наиболее популярным |
| По рейтингу ↓ | От высокого рейтинга к низкому |
| По рейтингу ↑ | От низкого рейтинга к высокому |
| По дате выхода ↓ | От новых к старым |
| По дате выхода ↑ | От старых к новым |
| По названию (А-Я) | Алфавитный порядок |
| По названию (Я-А) | Обратный алфавитный порядок |

При изменении сортировки каталог обновляется автоматически.

### Количество фильмов на странице

Используйте выпадающий список **«Показывать»** для выбора количества фильмов, отображаемых на одной странице:

- 5 фильмов
- 10 фильмов (по умолчанию)
- 15 фильмов
- 20 фильмов

### Пагинация

Для навигации по страницам каталога используются элементы пагинации в нижней части страницы:

- **Кнопка «← Назад»** — переход на предыдущую страницу (неактивна на первой странице)
- **Номера страниц** — прямой переход к конкретной странице
- **Текущая страница / Всего страниц** — информационный блок
- **Кнопка «Вперед →»** — переход на следующую страницу (неактивна на последней странице)

При большом количестве страниц отображаются не все номера, а только ближайшие к текущей, с многоточием для пропущенных.

### Кнопка обновления

Кнопка **«Обновить»** (с иконкой круговой стрелки) позволяет принудительно перезагрузить данные с сервера без изменения текущих параметров сортировки и поиска.

## Просмотр детальной информации

Для перехода к детальной информации о фильме **кликните по карточке фильма** (по любой её области, кроме кнопки удаления).

### Содержание детальной страницы

Детальная страница отображает всю доступную информацию о фильме:

**Основная информация:**
- Постер фильма (увеличенный)
- Фоновое изображение
- Название и оригинальное название
- Слоган (если есть)
- Дата выхода
- Продолжительность
- Рейтинг и количество голосов
- Жанры

**Описание:**
- Полный текст сюжета

**Съёмочная группа:**
- Режиссёр
- Актёрский состав (до 10 человек)

**Производственная информация:**
- Кинокомпании
- Страна производства
- Бюджет
- Сборы

**Внешние ссылки (при наличии):**
- **Смотреть трейлер** — ссылка на YouTube
- **IMDb** — страница фильма на Internet Movie Database
- **Официальный сайт** — официальный веб-сайт фильма

### Возврат к каталогу

Для возврата к каталогу используйте:
- Кнопку «Назад» в браузере
- Ссылку «Главная» в навигационном меню
- Клик по логотипу в шапке сайта

## Добавление фильма

Приложение позволяет добавлять собственные фильмы в каталог. Добавленные фильмы сохраняются в локальном хранилище браузера и отображаются вместе с фильмами из API.

### Переход к форме добавления

Нажмите на ссылку **«Добавить фильм»** в навигационном меню в шапке сайта.

### Заполнение формы

Форма содержит следующие поля:

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| Название фильма | Текст | Да | Название фильма |
| Описание | Многострочный текст | Да | Сюжет фильма |
| Дата выхода | Дата | Да | Дата премьеры (не позднее текущего года) |
| Продолжительность | Число | Да | В минутах (1-500) |
| Жанры | Текст | Да | Через запятую (например: Боевик, Драма) |
| Режиссёр | Текст | Да | Имя режиссёра |
| Актёры | Текст | Да | Через запятую (например: Брэд Питт, Анджелина Джоли) |
| Страна | Текст | Да | Страна производства |
| Язык | Текст | Да | Язык фильма |
| URL постера | URL | Нет | Ссылка на изображение |
| Рейтинг | Число | Нет | Оценка от 0 до 10 |
| Бюджет | Число | Нет | В долларах США |

### Дополнительные поля

После заполнения основных полей и нажатия кнопки «Добавить фильм» система последовательно запросит:

1. **Ссылку на трейлер** (YouTube) — опционально
2. **Ссылку на IMDb** — опционально
3. **Ссылку на официальный сайт** — опционально

Для пропуска любого из этих полей нажмите «Отмена» в диалоговом окне.

### Валидация

Перед сохранением форма проходит валидацию:
- Все обязательные поля должны быть заполнены
- Название должно содержать минимум 2 символа
- Описание — минимум 10 символов
- Продолжительность — положительное число

При обнаружении ошибок они будут перечислены в модальном окне.

### Сохранение

После успешной валидации:
1. Отображается индикатор загрузки
2. Данные сохраняются в локальное хранилище браузера
3. Показывается модальное окно с подтверждением
4. Форма очищается для возможности добавления следующего фильма

Добавленный фильм появится в общем каталоге на главной странице.

## Удаление фильма

Удалить можно только фильмы, добавленные пользователем. Фильмы из TMDB API удалить невозможно.

### Как отличить пользовательский фильм

Пользовательские фильмы имеют специальную кнопку удаления (иконка корзины), которая появляется в левом верхнем углу карточки при наведении курсора.

### Процедура удаления

1. Наведите курсор на карточку фильма, который хотите удалить
2. Нажмите на появившуюся кнопку с иконкой корзины
3. В появившемся диалоговом окне подтвердите удаление, нажав «ОК»
4. Фильм будет удалён из каталога, и список автоматически обновится

**Внимание:** Удаление необратимо. Восстановить удалённый фильм невозможно.

## Мобильная версия

Приложение полностью адаптировано для работы на мобильных устройствах.

### Особенности мобильной версии

**Навигационное меню:**
- На узких экранах меню сворачивается в бургер-кнопку (три горизонтальные линии)
- Для открытия меню нажмите на бургер-кнопку
- Повторное нажатие закрывает меню

**Сетка фильмов:**
- На мобильных устройствах карточки располагаются в один столбец
- Карточки занимают всю доступную ширину экрана

**Панель управления:**
- Элементы сортировки и выбора количества располагаются вертикально
- Кнопка обновления перемещается под основные элементы

**Детальная страница:**
- Постер и информация располагаются в одну колонку
- Постер центрируется и занимает комфортную ширину

**Форма добавления:**
- Поля формы растягиваются на всю ширину
- Сдвоенные поля (в одной строке) перестраиваются в один столбец

## Часто задаваемые вопросы

### Почему не загружаются фильмы?

**Возможные причины:**
1. Отсутствует подключение к интернету
2. Проблемы с API ключом TMDB
3. Превышен лимит запросов к API

**Решение:**
- Проверьте подключение к интернету
- Обновите страницу
- Если проблема сохраняется, обратитесь к администратору

### Почему не отображается постер фильма?

Некоторые фильмы в базе TMDB могут не иметь постера. В этом случае отображается стандартное изображение-заглушка.

Для пользовательских фильмов:
- Убедитесь, что указана корректная ссылка на изображение
- Ссылка должна начинаться с http:// или https://

### Как долго хранятся добавленные мной фильмы?

Добавленные фильмы хранятся в локальном хранилище вашего браузера. Они сохраняются до тех пор, пока:
- Вы не удалите их вручную
- Не очистите данные браузера
- Не переустановите браузер

Фильмы **не синхронизируются** между разными устройствами или браузерами.

### Можно ли редактировать добавленный фильм?

В текущей версии приложения редактирование добавленных фильмов не поддерживается. Если нужно изменить информацию, удалите фильм и добавьте его заново с правильными данными.

### Почему кнопка «Вперед» неактивна?

Кнопка «Вперед» становится неактивной, когда вы находитесь на последней странице каталога. Аналогично, кнопка «Назад» неактивна на первой странице.

### Что делать, если приложение работает медленно?

**Рекомендации:**
1. Уменьшите количество фильмов на странице (выберите 5 или 10)
2. Проверьте скорость интернет-соединения
3. Закройте другие вкладки браузера, потребляющие ресурсы
4. Обновите страницу
